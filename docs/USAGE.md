# Using the ProActive Design System

## CSS
Use `dist/css/tokens.css` to get CSS variables.

## Webflow
Recommended: export `dist/css/tokens.css` into a global embed.

## React
Consume `dist/json/tokens.flat.json` or create a small wrapper that maps token names to CSS variables.

## Assets
Logos live under `new_tokens/assets/logos/`.
Fonts live under `new_tokens/fonts/`.
